## Interact Budget: Start Visualizing Your XML Data
Experience the power of efficient XML data analysis with Interact Budget. Unlock the potential of  XML data with seamless integration of frontend and backend technologies. From parsing to visualization, Interact Budget has everything need to streamline data analysis workflow and make informed decisions.
[Preview Coming Soon], [Source Code Frontend](https://github.com/bforbilly24/final-project-nextjs-frontend-app), [Source Code Backend](https://github.com/bforbilly24/final-project-laravel-frontend-app).

### Features
- **Parsing XML Data:** EXML data is parsed into JSON format using Laravel, leveraging its robust parsing capabilities and seamless integration with PostgreSQL for efficient data storage.
- **Data Visualization:** Visualized data is fetched and displayed in Next.js, providing users with an intuitive and interactive interface for analyzing XML data.
- **PDF Download and Table View:** Users can download visualized data in PDF format or view it in a table format, allowing for easy access and analysis of parsed XML data.
- **Customizable View::** The show/hide filter feature enables users to customize their view by selecting or deselecting specific columns, ensuring a personalized and efficient data analysis experience.
