## Yayasan Syech Maulana Umar Masud

Comprehensive educational foundation platform serving as the digital hub for Yayasan Syech Maulana Umar Masud's educational excellence and community engagement. This sophisticated content management system showcases foundation activities, achievements, and institutional progress while providing seamless information access for all stakeholders.

### Features

- **Advanced Content Management System:** Sophisticated CMS architecture enabling seamless content sourcing, organization, and publication of institutional content, ensuring stakeholders have access to current and relevant information about foundation activities and educational programs.

- **Dynamic Article Management:** Complete CRUD operations for news articles, announcements, and educational insights with intuitive admin interface, enabling efficient content creation and management for timely community communication.

- **Comprehensive Activity Documentation:** Professional photo management system for documenting foundation events, educational initiatives, and community engagements with organized galleries that provide visual storytelling of institutional milestones and achievements.

- **Multi-Institution Profile Management:** Detailed institutional data management with CRUD operations for maintaining comprehensive information including institution details, leadership profiles, achievement records, staff directories, and social media integration.

- **Intelligent Monthly Program Gallery:** Dynamic photo showcase system with automated monthly content display, featuring smart slider functionality that highlights current month's work programs and seasonal activities based on calendar integration.

- **Achievement Showcase Portal:** Dedicated visual documentation system for celebrating foundation accomplishments, awards, and success stories that inspire community engagement and demonstrate institutional excellence and educational impact.

- **Real-Time News Portal:** Advanced news management system providing immediate updates on foundation activities, educational developments, and community announcements through user-friendly content interface and automated distribution.

- **Scalable Multi-Institution Architecture:** Comprehensive management framework supporting multiple educational institutions under the foundation umbrella, each with dedicated profiles, achievement tracking, and documentation systems.

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications with server-side rendering and static site generation
- [TypeScript](https://www.typescriptlang.org) - Typed superset of JavaScript providing enhanced development experience and code reliability
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework for rapid UI development and consistent design systems
- [ShadcnUI](https://ui.shadcn.com) - Modern component library providing accessible and customizable UI components built with Radix UI
