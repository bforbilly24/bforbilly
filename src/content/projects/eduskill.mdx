## Eduskill Landing Page

Premier destination for unlocking educational potential and achieving learning excellence. This engaging landing page serves as the gateway to comprehensive educational services, combining cutting-edge technology with innovative design to showcase transformative learning opportunities and inspire educational success.

### Features

- **Compelling Hero Section:** Powerful introduction with clear value proposition, engaging visuals, and strong call-to-action elements that immediately communicate the platform's educational benefits and encourage user engagement.

- **Course Showcase:** Strategic presentation of available courses and learning paths, highlighting key features, benefits, and outcomes to help visitors understand the educational value and make informed decisions.

- **Animated User Experience:** Smooth scroll animations and interactive elements powered by AOS library, creating an engaging and memorable browsing experience that keeps visitors engaged throughout their journey.

- **Success Stories:** Testimonials and achievement highlights that build trust and credibility, showcasing real student outcomes and the transformative impact of the educational platform.

- **Modern Design System:** Clean, professional aesthetic with consistent branding, typography, and color schemes that reflect educational excellence and technological innovation.

- **Mobile-Responsive Layout:** Optimized viewing experience across all devices, ensuring accessibility and usability for prospective students regardless of how they access the platform.

### Stack

- [React](https://react.dev) - Modern JavaScript library for building dynamic user interfaces with component-based architecture
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework for rapid UI development and consistent design systems
- [AOS](https://michalsnik.github.io/aos) - Animate On Scroll library for smooth CSS3 animations and enhanced user experience
