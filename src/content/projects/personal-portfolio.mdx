## Portfolio: Personal Detail Information
A portfolio website serves several functions, primarily focused on showcasing an individual's or a company's work, skills, and accomplishments. Key functions of a portfolio website like showcasing work, highlighting skills and expertise, building credibility, attracting opportunities, providing contact information, personal branding.

Overall, a portfolio website serves as a versatile tool for showcasing work, attracting opportunities, and building a professional online presence. Its specific functions may vary depending on the goals and objectives of the individual or company behind it.
[Preview](https://halim-portfolio-old.vercel.app/), [Source code](https://github.com/bforbilly24/personal-portfolio-next-frontend-app).

### Stack
- [Next.js](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
